Programa

Escribe JavaScript para enseñarle a tu guerrero qué hacer dependiendo de la situación. Selecciona habilidades para personalizar cómo juega tu guerrero.

Juega

Lanza el juego desde tu terminal y comprueba cómo le va a tu guerrero.

Inicio Rápido

  1. Instalar WarriorJS:
    npm install --global @warriorjs/cli
  2. Iniciar el juego:
    warriorjs
  3. ¡Crea a tu guerrero y comienza tu viaje!
