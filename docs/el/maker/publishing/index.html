Δημοσίευση

Αυτή είναι η ελάχιστη δομή ενός πακέτου πύργου:

warriorjs-tower-got
├── index.js
└── package.json

Όπου το index.js θα περιέχει τον κώδικα που γράψαμε σε αυτόν τον οδηγό, και package.json το πακέτο πληροφοριών npm:

{
  "name": "warriorjs-tower-got",
  "version": "0.1.0",
  "description": "There is only one war that matters: the Great War. And it is here.",
  "main": "index.js",
  "keywords": ["warriorjs-tower"],
  "dependencies": {
    "@warriorjs/geography": "^0.4.0"
  }
}

Μερικές ειδικές επισημάνσεις:

  • Το όνομα πακέτου πρέπει να ξεκινά με warriorjs-tower- για να φορτωθεί αυτόματα ο πύργος στο WarriorJS.
  • Το warriorjs-tower θα πρέπει να είναι στα "keywords" για ευκολότερη εύρεση του πύργου σου.

Όταν δουλεύεις με έναν πύργο μπορείς να χρησιμοποιήσεις το πακέτο npm για να δημιουργήσεις ένα αρχείο tar για αυτό, και έπειτα να το εγκαταστήσεις όπου εγκατέστησες το @warriorjs/cli με την εντολή:

npm install <path/to/tarball>

Μόλις το κάνεις, το warriorjs θα πρέπει να φορτώσει τον πύργο σου αυτόματα.

Αφού ελέγξεις και προσαρμόσεις τον πύργο σου, είσαι έτοιμος να τον δημοσιεύσεις στο npm για να παίξουν και άλλοι. Ακολούθησε αυτόν τον οδηγό για να μάθεις πως δημοσιεύουμε ένα πακέτο στο npm.
