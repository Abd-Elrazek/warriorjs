Configuració

Comencem per instal·lar WarriorJS globalment amb npm.

Obriu el terminal i executeu:

npm install --global @warriorjs/cli

IMPORTANT: Node.js >=8 necessita instal·lar-se al vostre ordinador abans d'executar l'ordre npm. El mètode d'instal·lació recomanat és a través del instal·lador oficial.

Després d'instal·lar el joc, podeu executar-lo executant l'ordre `warriorjs</ code> a la terminal:

warriorjs
`

Això és! Això us guiarà a través de la creació del vostre guerrer. Doneu la vostra guerrer un nom propi i triar la torre d'inici.

Després de fer això, haureu de tenir la següent estructura de fitxers (hem decidit per nomenar el nostre guerrer després del fill bastard de Lord Eddard Stark):

warriorjs
└── jon-snow-beginner
    ├── Player.js
    └── README.md
  • jon-snow-beginner és el directori del vostre perfil.
  • Player.js és el cervell del teu guerrer, estaràs editant aquest fitxer sovint.
  • README.md conté les instruccions per al nivell actual.

Continueu i obriu README.md per trobar les instruccions del primer nivell.
