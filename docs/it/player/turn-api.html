API del turno

Il metodo playTurn in Player.js riceve un istanza del turno del tuo guerriero. I metodi che puoi richiamare in quel turno sono determinati dalle abilità che il tuo guerriero ha a disposizione nel livello attuale. Vedi il file README nella cartella del tuo profilo per scoprire quali sono.

Ecco un esempio estratto dal README del secondo livello della torre "BabySteps":

### Abilities

#### Actions

- `warrior.attack()`
- `warrior.walk()`

#### Senses

- `warrior.feel()`

In questo livello, il tuo guerriero ha a disposizione le abilità "attaccare", "percepire" e "camminare", il che significa che puoi richiamare questi tre metodi durante il tuo turno: warrior.attack(), warrior.feel(), e warrior.walk().

Molte abilità possono essere eseguite nelle seguenti direzioni: "frontale", "dietro", "sinistra", e "destra". Devi inviare una stringa che abbia come primo argomento la direzione, ad es. warrior.walk('backward').
