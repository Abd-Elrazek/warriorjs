Opzioni

Ci sono varie opzioni che puoi passare al comando warriorjs per personalizzare il gioco. Puoi eseguire warriorjs --help per visualizzare tutte le opzioni disponibili.

Ecco qui una lista dettagliata:

--directory <path>

Percorso per una directory sotto la quale eseguire il gioco. Da impostazione predefinita, viene utilizzata la directory attuale di lavoro.

--level <number> (solo modalità epica)

Esercitati con un livello. Utilizzare questa opzione per i livelli che non riesci a superare oppure per quelli per cui desideri migliorare il tuo punteggio.

--silent

Sopprimere il registro della partita. Usa questa opzione se ti interessa solo l'esito di un livello e non di ogni fase della giocata.

--time <secondi>

Ritardare ogni turno di un certo numero di secondi. Per impostazione predefinita, ogni passo di ogni turno è ritardato di 0,6 secondi.

--yes

Assume una risposta positiva nei dialoghi di conferma non distruttivi.
