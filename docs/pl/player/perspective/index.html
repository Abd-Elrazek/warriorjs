Perspektywa

Mimo tego, że jest to gra tekstowa, wyobraź ją sobie jako dwuwymiarową, gdzie widzisz podłogę poziomu z góry.

Każde piętro jest zawsze w kształcie prostokąta i składa się z szeregu kwadratów. Na jego krawędziach są ściany; nie możesz przez nie przejść. Możesz poruszać się po wszystkich kwadratach, w tym schodach, które nie są zajęte przez inną jednostkę (na jednym polu nie mogą stać dwie jednostki jednocześnie).

Oto przykład mapy piętra oraz legenda:

╔════╗
║C s>║
║ S s║
║C @ ║
╚════╝

> = stairs
@ = Jon Snow (20 HP)
s = Sludge (12 HP)
S = Thick Sludge (24 HP)
C = Captive (1 HP)
