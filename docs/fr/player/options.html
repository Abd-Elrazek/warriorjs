Options

Il y a plusieurs options que vous pouvez faire warriorjs pour personnaliser le jeu. vous pouvez exécuter warriorjs --help pour voir toutes les options disponibles.

Ci-dessous la liste détaillée:

--directory <chemin>

Accès au répertoire dans lequel exécuter le jeu. Par défaut, le répertoire de travail actuel est utilisé.

--level <nombre> (mode épique seulement)

Niveau de pratique. Utilisez cette option lorsque vous rencontrez des difficultés ou vous souhaitez améliorer votre score.

--silent

Supprimez "play log". Utiliser cette option si vous vous préoccupez juste de la fin de jeu d'un niveau et non de chaque étape du jeu.

--time <secondes>

La retard de chaque tour par secondes. Par défaut chaque étape de chaque tour est en retard de 0.6 secondes.

--yes

Appuyez oui dans une boîte de dialogue de confirmation non-destructive.
