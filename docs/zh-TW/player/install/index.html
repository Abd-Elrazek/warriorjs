Install

要開始遊戲，我們首先需要以npm在全域網絡安裝WarriorJS。

打開終端機及執行：

npm install --global @warriorjs/cli

注意： 在運行npm之前，你需要先在電腦安裝 Node.js >=8， 建議的安裝方法可以參考官方安裝頁面。

遊戲安裝之後，你可以在終端機執行warriorjs指令去運行遊戲：

warriorjs

安裝完成了！遊戲將引導你創造你的勇士，請幫你的勇士配上合適的名字，並選擇新手塔 (baby steps tower) 開始遊戲。

當你完成以上指示後，你應該得到以下的檔案結構 (例子中的勇士以艾德・史塔克的私生子命名)：

warriorjs
└── jon-snow-baby-steps
    ├── Player.js
    └── README.md
  • jon-baby-steps 是你設定檔的目錄。
  • Player.js 是你勇士的大腦，你將會經常編輯這個檔案。
  • README.md 含有現時塔層的指示。

現在讓我們繼續前進，打開README.md並查看第一層的指示。
