Věže

tower je světem WarriorJS. Krom definování pater můžou věže do hry přidávat také nové schopnosti, efekty a jednotky.

WarriorJS CLI ships with an entry-level tower built-in. You'll need to install any additional tower you want to play.

Instalace věží

Věže jsou automaticky nahrány pokud je máš nainstalované ve stejném node_modules adresáři jako @warriorjs/cli. To znamená, že pokud máš hru nainstalovanou globálně, bude muset nainstalovat další věže také globálně. Na druhou stranu, pokud hraješ hru z lokální instalace, budeš muset nainstalovat další věže lokálně.

Jména packages věží začínají @warriorjs/tower- pro oficiální věže a warriorjs-tower- pro komunitní věže.

Oficiální věže

  • @warriorjs/tower-baby-steps
  • @warriorjs/tower-tick-tick-boom (beta)
