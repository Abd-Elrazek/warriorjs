Obecné tipy

  • Pokud se někdy zasekneš na nějakém patře, přečti si pečlivě README a ujisti se, že jsi vyzkoušel všechny dostupné abilities.

  • Pokud ztrácíš zdraví moc rychle, odpočívej, pokud není v okolí žádný nepřítel (zatímco sleduješ své zdraví). Also, try to use far-ranged weapons whenever possible (such as the bow in the "Baby Steps" tower).

  • Senses jsou levné, tak je hojně používej Ukládej si informace ze senses, aby Ti pomohly udělat rozhodnutí, jaké actions jsou ty správné v budoucnu.

  • Pokud Ti jde o body, nezapomeň vyčistit celou oblast. I když jsi blízko schodů, nechoď nahoru dokud jsi nezískal vše (pokud máš dostatek zdraví). Use far-ranged senses (such as "look" and "listen" in the "Baby Steps" tower) to determine if there are any enemies left.
