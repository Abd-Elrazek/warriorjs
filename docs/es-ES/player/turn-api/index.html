API del Turno

El método playTurn en Player.js recibe como parámetro una instancia del turno de tu guerrero. Los métodos que puedes llamar sobre ese turno están determinados por las habilidades que tu guerrero tiene disponibles en el nivel actual. Consulta el archivo LÉEME en el directorio de tu perfil para saber cuáles son.

Aquí hay un ejemplo extraído del archivo LÉEME del segundo nivel en la torre "Baby Steps":

### Abilities

#### Actions

- `warrior.attack()`
- `warrior.walk()`

#### Senses

- `warrior.feel()`

En este nivel, el guerrero tiene las habilidades "attack", "feel" y "walk", lo que significa que puede llamar a estos tres métodos en su turno: warrior.attack(), warrior.feel() y warrior.walk().

Muchas habilidades pueden ser realizadas en las siguientes direcciones: adelante ("forward"), atrás ("backward"), izquierda ("left") y derecha ("right"). Tienes que pasar una cadena con la dirección como el primer argumento, por ejemplo warrior.walk('backward').
