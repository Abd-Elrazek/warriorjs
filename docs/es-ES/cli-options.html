Opciones de la CLI

Existen varias opciones que puedes pasarle al comando warriorjs para personalizar el juego. Puedes ejecutar warriorjs --help para ver todas las opciones disponibles.

Aquí hay una lista detallada:

--directory <ruta>

Ruta a un directorio bajo el cual al ejecutar el juego. De forma predeterminada, se utiliza el directorio de trabajo actual.

--level <número> (solo modo épico)

Practica un nivel. Utiliza esta opción en niveles en los cuales estés teniendo dificultad o quieras afinar la puntuación.

--silent

Suprime el log del juego. Utiliza esta opción si solo te interesa el resultado de jugar un nivel, y no cada paso de la jugada.

--time <segundos>

Retrasa cada turno una cierta cantidad de segundos. De forma predeterminada, cada paso de cada turno se retrasa 0,6 segundos.

--yes

Asume una respuesta afirmativa en los diálogos de confirmación no destructivos.
