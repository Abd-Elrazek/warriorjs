開始遊戲

從你的命令列終端啟動遊戲, 並檢查你的勇士。

快速入門

  1. 安裝 WarriorJS:
    npm install --global @warriorjs/cli
  2. 啟動遊戲:
    warriorjs
  3. 創造你的勇士, 開始你的偉大旅程!
