Kod

Savaşçınıza bulunduğu durumda ne yapması gerektiğini öğretmek için JavaScript kodu yazın. Savaşçınızın oynayışını kişileştirmek için yetenekler seçin.

Oyun

Oyunu terminalinizden başlatın ve savaşçınızın durumunu kontrol edin.

Hızlı Başlangıç

  1. WarriorJS'i kurun:
    npm install --global @warriorjs/cli
  2. Oyunu başlatın:
    warriorjs
  3. Savaşçınızı oluşturun ve yolculuğunuza başlayın!
